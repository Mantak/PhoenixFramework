# PhoenixFramework 中文入门教程

我本前端开发。

过去你可能在网上看过不少人喷前端开发人员的，说门槛太低，阿猫阿狗都敢自称工程师的。

以后你可能要看到喷 PhoenixFramework 开发的，原因还是因为门槛太低。

可是，PhoenixFramework 真的太容易上手了。

不信你也来试试。

教程中使用的软件版本号如下：

1. Elixir - 1.3.4
2. PhoenixFramework - 1.2.1
3. Ecto - 2.0.5
